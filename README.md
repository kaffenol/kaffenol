# Kaffenol

Kaffenol is a planned cross-platform image editing environment focused on approachable everyday editing, progressive disclosure of advanced tools, and a command-driven architecture suitable for automation.

The project is currently in the **repository and architecture scaffolding phase**. There is intentionally no application code yet.

## Intended technology

- Kotlin Multiplatform
- Compose Multiplatform for application UI
- Skia through Skiko for canvas display and compositing
- Platform-independent Kotlin modules for documents and image processing
- Optional native or Rust acceleration only after profiling demonstrates a need
