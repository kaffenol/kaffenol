# Product vision

Kaffenol will be an approachable, cross-platform image editing environment that lets beginners improve ordinary images without first learning professional imaging terminology, while allowing experienced users to reveal deeper controls as their needs grow.

Its interaction model will borrow the discoverability and command-driven extensibility of modern code editors: menus, shortcuts, toolbars, and a command palette invoke the same underlying actions. The canvas remains central, with adaptable panels and layouts for desktop and touch devices.

Kaffenol should grow from safe JPEG/PNG editing into non-destructive compositing, colour-managed technical imaging, and eventually headless automation. Each stage must remain coherent and useful; later ambitions must not distort the first usable release.

## Initial product boundary

The first serious version targets desktop and Android, with iOS considered later. Web is not an initial objective. The first capability milestone covers opening and exporting common web formats, crop, global exposure and colour adjustments, and undo history.

No implementation decisions beyond the accepted architecture records should be treated as final during the scaffold phase.
