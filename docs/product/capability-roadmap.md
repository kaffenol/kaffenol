# Capability roadmap

The roadmap follows progressive levels from the source planning notes. Names here describe capability stages, not release numbers or commitments.

1. **Basics:** JPEG/PNG open and export, crop with aspect locking, global exposure/contrast/colour controls, and simple undo.
2. **Intermediate control:** histograms, curves and levels, RAW ingestion, local adjustments, healing/cloning, and basic selections.
3. **Advanced compositing:** robust layers, masks, blend modes, targeted colour grading, advanced selections, pen tools, and improved noise reduction/sharpening.
4. **Technical imaging:** high bit depth, explicit colour spaces and gamma, deep RAW controls, ICC management, soft proofing, LUTs, and metadata preservation.
5. **Automation:** batch and headless execution, stable scripting/API integration, metadata tooling, and extensible processing graphs.

Advancing to a stage requires a focused plan and explicit architecture decisions. Later-stage needs should influence extensibility, not cause speculative implementation now.
